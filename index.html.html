Construindo segurança com código

Acredito que a melhor forma de aprender segurança é colocando a mão na massa. Por isso desenvolvo projetos práticos que simulam desafios reais de um Centro de Operações de Segurança.

Meu objetivo é construir uma carreira sólida em Segurança da Informação — desenvolvendo soluções que auxiliem na detecção de ameaças, automatização de processos e fortalecimento da postura de segurança das organizações.

Além do estudo contínuo, utilizo plataformas como TryHackMe para reforçar conceitos em laboratórios práticos, com foco em análise de alertas, resposta a incidentes e o framework MITRE ATT&CK.

Ferramentas desenvolvidas

SOC IOC Triage Automator
Em desenvolvimento

Ferramenta Python para automatizar a triagem inicial de Indicadores de Comprometimento (IoCs). Simula o fluxo de trabalho de um analista SOC — da extração ao relatório final — reduzindo o tempo de resposta em análises de segurança.

  • Extração e validação de IoCs (IPs, domínios, hashes e URLs)
  • Defanging automático de URLs para manipulação segura
  • Enriquecimento com WHOIS e integração com a API do VirusTotal
  • Classificação de risco e score de ameaça por indicador
  • Mapeamento de possíveis técnicas do framework MITRE ATT&CK
  • Geração automática de evidências em formatos TXT e JSON
Python VirusTotal API WHOIS MITRE ATT&CK REST APIs JSON FastAPI
